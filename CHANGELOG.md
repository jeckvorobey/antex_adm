# Changelog

Все значимые изменения в проекте AntEx Admin. Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [Unreleased]

### Added

- Страница управления ставками AEX (AexRatesSettingsPage): глобальная ставка, персональные и партнёрские ставки с инлайн-редактированием
- Страница кошельков AEX (AexWalletsPage): таблица кошельков с поиском и отображением баланса
- Журнал операций AEX (AexJournalPage): фильтры, пагинация, отображение всех операций
- Страница ручных операций AEX (AexManualOpsPage): форма зачисления/списания с подтверждением
- Подменю AEX в главном навигационном меню (MainLayout) с раскрывающимся списком
- Маршруты: /aex/rates, /aex/wallets, /aex/journal, /aex/manual-ops
- Кнопка «Сгенерировать коды» на странице пользователей для batch-генерации referral_code
- POST /api/admin/aex/generate-referral-codes — вызов при нажатии кнопки с confirm dialog
- Автообновление таблицы пользователей после успешной генерации кодов
- 36 новых тестов для AEX-страниц и генерации реферальных кодов

### Changed

- UsersPage: добавлена видимость кнопки генерации только при наличии пользователей с referral_code IS NULL
